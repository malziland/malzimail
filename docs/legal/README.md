## Warum das wichtig ist

Bei jeder Datenschutzbehörden-Anfrage, Auskunftsanfrage oder einem Anwaltsschreiben
musst du als Betreiber nachweisen können, dass die eingesetzten Drittanbieter
(hier: Cloudflare) die DSGVO-konforme Datenverarbeitung vertraglich zugesichert haben.
Diese PDF ist der Nachweis für Cloudflare.

## Pflege

Cloudflare aktualisiert das DPA gelegentlich. Wenn das passiert, taucht eine neue
Version auf der Cloudflare-Seite auf (höhere Versionsnummer und neueres Datum).
Dann diese Datei durch die aktuelle ersetzen und die Tabelle oben aktualisieren.
